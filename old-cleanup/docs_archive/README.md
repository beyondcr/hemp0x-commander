# Hemp0x App Documentation

This folder documents the Hemp0x App (formerly "Commander") and how to build,
run, and extend it. The project is a cross-platform GUI built with Tauri and
Svelte, designed to control the Hemp0x daemon and wallet with a modern UI.

Docs index:
- docs/OVERVIEW.md: Product goals, sections, and UI flow.
- docs/ARCHITECTURE.md: System design and component boundaries.
- docs/DEVELOPMENT.md: Dev setup, run commands, and troubleshooting.
- docs/FEATURES_COMMANDER.md: Commander feature map and parity with legacy.
- docs/ROADMAP.md: Planned features and major milestones.
